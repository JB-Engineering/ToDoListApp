A simple Android To‑Do List app demonstrating CRUD operations with Room, MVVM, and RecyclerView.

Features
- Add, edit, delete tasks  
- Room database for persistence  
- MVVM architecture  
- RecyclerView list with swipe‑to‑delete  
- Material 3 UI  

Tech
-Java 
-Room 
-LiveData 
-MVVM
-RecyclerView 
-Material 3
